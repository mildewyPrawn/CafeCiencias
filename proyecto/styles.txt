Bueno por en si lo que hice fue :
En el html en el div donde esta el form
le puse un class para poder acceder a el en el css

<div class="bf-back">

Y en el css :
.bf-back{
  //blanco trasparente
  background: #ffffff6e;
  //el ancho del div
  width: 15%;
  //para centrar el div
  display: inline-block;
  //paddings para hacer mas alto el div
  padding-top: 50px;
  padding-bottom: 50px;
}

Tambien en los 3 href que habia debajo de la forma
los meti en un div para darles estilo

<div class="link-account">

En el css:
//esta es para el div en general
.link-account{
  font-size: 15px;
  color: #fff;
}

// este es para que los estilos se apliquen tambien a los href
.link-account a{
  color: #fff;
}

Ahora, para el pie de pagina siento que estaba muuuy arriba
entonces lo puse más abajo de esta forma:
- Primero le puse un class al div que lo contenia
- lo modifique a h2 para poderle dar un estilo más

<div class="piep">
  <h2 > Copyright © 2020. Powered by </h2>
  <a href='https://www.143buzzgame.com/'>143 BuzzGame</a>
</div>

En el css :

//estilos el div
.piep {
padding-top: 15%;
color: #fff;
}

//estilos aplicados en href
.piep a {
font-size: 15px;
padding-top: 15%;
color: #fff;
}

Tambien agregue h2 al css:
h2 {
  font-size: 15px;
}


Y en cuanto el boton de inciar sesión solo con el nombre
de la clase del boton lo mande a llamar en css y le puse
las caracteristicas deseadas

.btn{
  background-color: #C1B20D;
}


Ahora
Recomiendo muchisimo siempre siempre ponerle nombre de clase
a los div, containers, label, etc.
de esta forma en el css todo es mucho más personalizable
y en si es algo obligatorio por que si no se hace un cagadero.
